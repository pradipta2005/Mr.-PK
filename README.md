## About

This is the source code of my personal portfolio — an immersive, cinematic single-page experience built entirely with vanilla HTML, CSS, and JavaScript. No frameworks, no component libraries. Every animation and interaction was hand-crafted to reflect how I think about craft: deliberately, with attention to detail.

The site doubles as a live résumé and a demonstration of what's possible on the open web without reaching for a framework.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5, Semantic Markup |
| Styling | CSS3 — Custom Properties, Grid, Flexbox, Clip-path Animations |
| Animation Engine | GSAP 3 + ScrollTrigger + ScrollToPlugin |
| Scroll | Lenis (smooth inertia scrolling) |
| Physics | Matter.js (skills bubble simulation) |
| 3D / WebGL | Three.js r134 (interactive footer globe) |
| Typography | SplitType.js · Instrument Serif · Manrope |
| Deployment | GitHub Pages |

## Sections

**Hero** — Full-screen canvas with a particle/WebGL intro and a cinematic reveal sequence.

**Selected Works** — Horizontal scroll gallery showcasing 8 projects across data engineering, ML, analytics, and frontend development.

**The Architect** — Brief personal background with a liquid-distortion portrait effect.

**Proficiencies** — Physics-based skill bubbles using Matter.js for a gravity simulation.

**Experience** — Organic SVG timeline featuring 3 internship roles.

**Academics** — Sticky parallax card stack for education history.

**Certifications** — Interactive accordion with certificate image previews on hover.

**Contact / Footer** — Three.js rotating globe, live IST clock, magnetic hover links.

## Running Locally

No build tools required. Clone the repo and open `index.html` in a browser.

```bash
git clone https://github.com/pradipta2005/Mr.-PK.git
cd Mr.-PK
# open index.html in your browser
```

> JavaScript must be enabled. The WebGL canvas and all animations depend on it.
